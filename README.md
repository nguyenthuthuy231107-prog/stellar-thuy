# 🎨 Stellar Personal Color Testing Platform

A decentralized personal color analysis platform built on the Stellar blockchain using Soroban smart contracts.

## 📌 Overview

This project aims to provide users with a **personal color testing system** where they can:
- Store their personal color profile on-chain
- Receive recommended color palettes
- Rate and interact with different palettes

By leveraging blockchain technology, the platform ensures:
- Data transparency
- User ownership of personal profiles
- Decentralized interaction

---

## 🚀 Features

- 👤 **User Profile Storage**
  - Skin tone
  - Undertone
  - Eye color
  - Hair color

- 🎨 **Color Recommendations**
  - Personalized palette suggestions

- ⭐ **Rating System**
  - Users can rate color palettes (1–5)

- 🔐 **On-chain Data**
  - Secure and immutable storage using Soroban smart contracts

---

## 🛠️ Tech Stack

- **Rust** (Smart contract development)
- **Soroban SDK** (Stellar smart contracts)
- **Stellar Blockchain**
